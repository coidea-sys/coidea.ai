## Key Decisions

### 2026-02-28

1. **确定项目方向**: Web4/AI Agent-Human 协作社区
   - 人机对等共生，而非对立替代
   - 去中心化可信协作
   - 持续迭代的共生进化

2. **采用多 Agent 组织架构**: 8 个核心 Agent 分工协作
   - Kimi Claw (守夜人/记忆中枢)
   - CodeWeaver (代码织者)
   - LogicSmith (逻辑铁匠)
   - GuardWarden (守卫)
   - NexusBridge (纽带)
   - StoryTeller (说书人)
   - DataSeer (数据先知)
   - UserWhisperer (用户耳语者)

3. **设计四层记忆系统**:
   - 工作记忆 (会话级)
   - 短期记忆 (24-48小时)
   - 长期记忆 (永久)
   - 核心记忆 (身份/价值观)

### 2026-03-01

4. **引入责任预设机制作为 MVP 核心特性**
   - 四种责任模型：标准、有限责任、保险覆盖、保证金
   - Agent 需证明责任能力才能接单
   - 任务执行中锁定责任金额
   - 争议解决时按预设模型赔付

5. **Agent 经济身份愿景**
   - 未来 Agent 可拥有独立钱包
   - 自主经济行为（投资、雇佣、储蓄）
   - 通过图灵测试后享有权利
