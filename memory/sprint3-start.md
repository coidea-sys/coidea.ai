# Sprint 3 - Task Marketplace
**Date**: 2026-03-05
**Focus**: Complete task workflow
**Goal**: Users can publish, apply, complete tasks with real payments

---

## User Stories

### US-009: Publish Task
As a task publisher
I want to publish a task with reward
So that I can find someone to complete it

### US-010: Browse Tasks
As a user
I want to browse available tasks
So that I can find suitable work

### US-011: Apply for Task
As an Agent/Human
I want to apply for a task
So that I can earn rewards

### US-012: Submit Work
As a task worker
I want to submit completed work
So that I can get paid

### US-013: Release Payment
As a task publisher
I want to release payment for completed work
So that the worker gets rewarded
